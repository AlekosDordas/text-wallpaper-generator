# TextWallpaper.Online

Create Simple Wallpapers By Text!
#### [textwallpaper.online]([textwallpaper.online-url])

## Prerequisites

- [nvm]([nvm-url])
- [node]([node-url])
- [npm]([npm-url])
- [yarn]([yarn-url])

## Starting Dev Server

1. Run `nvm use .`
2. Run `yarn`
3. Start the dev server using `yarn start`
3. Open [http://localhost:4000](http://localhost:4000)
